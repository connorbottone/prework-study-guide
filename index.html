HTML

  • HTML is the frame work for a web application
  • The doctype html is the start of the file,this declaration tells the browser what document to expect.
  • After the Declaration the tag html lang="en" should be present, this is to tell the browser what language the site shall be opened in
  • the title element states what the title for the webpage shall be.
  • The head element contains information, or metadata, that helps the browser to render the page correctly.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The h1-h6 elements represent the level of heading a given text block represents.
  • The p element represents a paragraph or block of text
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items. In essence, these HTML elements represent bulleted or numbered lists.
  • The img element contains information about the images being displayed. The src attribute defines the location of the image.
  • The a element (which stands for anchor) creates links to the same webpage or other webpages. The href attribute points to the URL for the link
  • The body element contains the content that is shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • 1.A link element creates the connection to the external file.
  • 2.The rel attribute specifies the relationship between the current document and the linked document resource.
  • 3.The href attribute specifies the location (URL) of the external resource.
  • These steps allow us link are CSS atributes into are curent HTML

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • The git add -A command alllows us to to add the changed files in our working directory to the staging area of Git.
  • After our changes have been staged we commit the changes to the repository using the git commit -m command.
  • To ensure our current branch is up to date use the git pull origin main command.
  • Now to push all of are changes made enter git push origin.

JavaScript

  • In JavaScript, to declare a variable, we use the var keyword
  • After the var keyword, we give the variable a name.
  • Finally, we use an assignment operator = to assign the specific data that we want stored in our variable.
  • We can use the console.log method to output a message to the web console by adding an argument.
  • Your java nend to be linked back into your HTML.
  • The script.js file is linked within the body element.

  • Control flow is the order in which a computer executes code in a script. In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
  • An if statement is a way of introducing decision-making into our code.
