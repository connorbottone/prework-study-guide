HTML

  • HTML is the frame work for a web application
  • The doctype html is the start of the file,this declaration tells the browser what document to expect.
  • After the Declaration the tag html lang="en" should be present, this is to tell the browser what language the site shall be opened in
  • the title element states what the title for the webpage shall be.
  • The head element contains information, or metadata, that helps the browser to render the page correctly.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The h1-h6 elements represent the level of heading a given text block represents.
  • The p element represents a paragraph or block of text
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items. In essence, these HTML elements represent bulleted or numbered lists.
  • The img element contains information about the images being displayed. The src attribute defines the location of the image.
  • The a element (which stands for anchor) creates links to the same webpage or other webpages. The href attribute points to the URL for the link
  • The body element contains the content that is shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
